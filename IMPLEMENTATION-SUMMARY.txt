USAGE EXAMPLES
================================================================================

Basic Usage:
  PanelLoader.attachToPanel('#heroPanel1').show();

With Options:
  PanelLoader.attachToPanel('#heroPanel1', {
      loaderType: 'spinner',
      loadingText: 'Loading magical content',
      duration: 1500,
      fadeInDuration: 600
  }).show();

Multiple Panels:
  const loaders = PanelLoader.attachToMultiplePanels(
      ['#panel1', '#panel2', '#panel3'],
      { loaderType: 'skeleton' }
  );
  loaders.forEach(l => l.show());
